A programming language is a formal, artificial language composed of a set of instructions used by developers to communicate
      with and control thebehavior of a machine, typically a computer.These languages define data and the operations that can be 
      performed on it, all with a precise syntax andsemantics that a computer can interpretand execute. 


   
Programs written in a human-readable high-level language are translated intomachine-readable binary code either by a compiler (ahead of time) or  an interpreter (at runtime).
